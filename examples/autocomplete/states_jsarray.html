AutoComplete Widget :: JS Array States

Welcome to AutoComplete.

DataSource

Two DataSource instances point to in-memory JavaScript arrays. The first array holds the 50 US states. The second array holds US area codes and their corresponding states for additional info.

AutoComplete

Since the two DataSources for this example are already loaded into memory via JavaScript arrays, queries should be very fast to return data. Therefore, both AutoComplete instances are configured to have a query delay of zero seconds. The second AutoComplete instance is configured to display two data fields in the container: the area code and the area code's state.
